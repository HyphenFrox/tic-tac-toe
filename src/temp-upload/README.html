Tic Tac Toe

A tic tac toe app that lets you rewind the game.

Technologies Used

  • SASS
  • React.js
  • MUI (Material UI)

How to run locally?

Clone this repository then run npm install to install all the dependencies and then run npm run start to start a local server.
